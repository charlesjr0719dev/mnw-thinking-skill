# 04 有效互動（Interacting Effectively）

## 在四大能力中的定位
密涅瓦四大核心能力（Cornerstones）構成一個由內而外的思考行動鏈：

1. **Thinking Critically**（批判性思考）—— 評估主張與證據
2. **Thinking Creatively**（創造性思考）—— 產生新點子
3. **Communicating Effectively**（有效溝通）—— 清晰傳達
4. **Interacting Effectively**（有效互動）—— **與他人共創結果**

前三者主要是「我與知識」「我與表達」的關係，**有效互動則是「我與他人、我與系統」的關係**——這是思考最終要落地的地方。再好的批判、創意、溝通，若無法在複雜人際與系統中實現，便停在象牙塔內。

## 四個子分類

### [[01-解決道德問題]]（3 個 HC）
倫理行動鏈：看見 → 判斷 → 行動。
涵蓋 [[道德考量]]、[[道德判斷]]、[[道德勇氣]]。

### [[02-複雜系統]]（6 個 HC）
系統分析工具箱：結構描繪 + 動態推演。
涵蓋 [[系統描繪]]、[[系統動力]]、[[多層次分析]]、[[網絡分析]]、[[複雜因果關係]]、[[湧現]]。

### [[03-談判和說服]]（5 個 HC）
影響力光譜：姿態、訊息、結構、誘因。
涵蓋 [[自信]]、[[談判]]、[[說服]]、[[助推]]、[[胡蘿蔔和棒子]]。

### [[04-與他人合作]]（8 個 HC）
合作三層：自我 → 對人 → 集體。
涵蓋 [[自我覺察]]、[[情商]]、[[一致性]]、[[責任感]]、[[個體差異]]、[[權力動態]]、[[領導原則]]、[[制訂策略]]。

## 學習路徑建議

### 路徑 A：由內而外（建議初學者）
1. 先紮根：[[自我覺察]] → [[情商]] → [[一致性]] → [[責任感]]
2. 再看他人：[[個體差異]] → [[權力動態]]
3. 學影響：[[自信]] → [[說服]] → [[談判]]
4. 上倫理：[[道德考量]] → [[道德判斷]] → [[道德勇氣]]
5. 看系統：[[系統描繪]] → [[複雜因果關係]] → [[系統動力]] → [[湧現]]
6. 進集體：[[領導原則]] → [[制訂策略]]

### 路徑 B：由外而內（職場應用者）
從最常用的 [[談判]]、[[說服]]、[[領導原則]] 入手，遇到困難時回溯到 [[情商]]、[[自我覺察]] 等基礎。

### 路徑 C：主題式
- 「如何領導」：[[領導原則]] + [[制訂策略]] + [[權力動態]] + [[情商]]
- 「如何影響」：[[說服]] + [[助推]] + [[談判]] + [[自信]]
- 「如何合作」：[[個體差異]] + [[情商]] + [[責任感]] + [[一致性]]
- 「如何面對複雜」：整組 [[02-複雜系統]] + [[制訂策略]]

## 練習建議
此能力層強調**情境實踐**。單讀概念無效，務必在真實互動中練習、回饋、修正。建議每週選 1 個 HC 做 7 天專注練習，並記錄三件事：用在哪、效果如何、下次調整。
