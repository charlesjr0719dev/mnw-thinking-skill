# 02-發掘與探索

## 子分類角色
「發掘與探索」是創意思考的**發現引擎**：在還不知道問題是什麼、答案長什麼樣時，你需要一組工具把混沌變成可研究的對象。這個子分類關注「怎麼從觀察資料裡看出模式、形成假說、建立模型」。

Minerva 把它放在 **Empirical Analyses** Cornerstone，是連接直覺與嚴謹研究方法的橋梁——先靠這組 HC 看見什麼值得研究，再用應用研究方法去驗證。

## HC 列表
- [[建立假說]]：把模糊的好奇心轉成可被資料打臉的具體陳述。
- [[資料視覺化]]：用對的視覺編碼讓眼睛看見表格看不見的模式。
- [[建模]]：把真實世界簡化成變數與關係，解釋現狀或預測未來。

## 學習建議路徑
1. 從 [[資料視覺化]] 開始，它是進入任何資料集的第一個動作。
2. 視覺化出現的模式會自然引發 [[建立假說]]。
3. 把多個假說整合成 [[建模]]，就能產生預測並與未來資料對照。
4. 循環：模型預測錯 → 重新視覺化殘差 → 修正假說 → 改模型。
