# 領導原則（#leadershipprinciples）

> 一句話定義：透過願景、賦能、決策與承擔，引導群體朝共同目標前進的原則與實踐。

## 密涅瓦怎麼定義
密涅瓦將此 HC 定義為：理解並應用有效領導的核心原則——願景建構、決策權衡、賦能與問責、情境領導——而非將領導視為單一人格特質。屬於 Working Effectively with Others 子能力。

## 核心概念
- **情境領導**（Hersey-Blanchard）：不同成熟度的下屬需不同風格
- **交易型 vs 變革型** 領導：管理現有 vs 創造新局
- **僕人式領導**（Greenleaf）：領導是服務而非支配
- **Level 5 Leadership**（Collins）：謙遜 + 專業意志
- **Extreme Ownership**（Willink）：領導承擔一切責任
- 領導不等於職位——沒有職位也能領導

## 典型應用場景
| 情境 | 怎麼用 |
|------|--------|
| 新團隊成立 | 先建立願景與行為準則，再談任務 |
| 危機時刻 | 清晰決策、承擔責任、透明溝通 |
| 高手成員 | 授權（委任式），不要微管理 |
| 新手成員 | 結構化指導（指示式），再逐步放手 |

## 常見誤用
- 把領導等同於「下命令」或「有職權」
- 固守單一風格，不隨情境調整
- 只領導上級看得見的績效，忽略團隊文化
- 過度追求「被喜歡」而不敢做艱難決定
- 把領導當成表演，缺少真實的承擔

## 如何自我練習
1. 每週回顧：我在哪個情境扮演了領導角色？效果如何？
2. 練習「跟隨者 → 領導者」切換，兩者都是技能
3. 讀《Good to Great》《Extreme Ownership》《The Leadership Challenge》
4. 主動爭取小規模領導機會，累積情境判斷

## 與其他 HC 的關係
- [[責任感]]：領導核心是承擔集體責任
- [[制訂策略]]：領導需設計可執行的策略
- [[權力動態]]：領導運作於權力結構中
- [[說服]]：願景傳播需說服力
- [[情商]]：讀懂團隊情緒是領導要件
- [[自我覺察]]：知道自己的領導盲點
