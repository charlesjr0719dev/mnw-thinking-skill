# 責任感（#responsibility）

> 一句話定義：主動承擔任務、結果與後果，不推諉、不委過。

## 密涅瓦怎麼定義
密涅瓦將此 HC 定義為：在個人與集體情境中主動承擔自己該負的職責，對行動結果負責，並能區分個人責任、共同責任與制度責任。屬於 Working Effectively with Others 子能力。

## 核心概念
- **Accountability vs Responsibility**：前者是對結果負責（對外），後者是對任務負責（對內）
- **個人 vs 集體責任**：集體責任不等於無人負責
- **事前 vs 事後**責任：承擔角色 vs 承擔後果
- 責任感包含「不是我的錯但我願意處理」的主動性
- **道德幸運**（moral luck）：同樣行為不同結果，責任如何分配？

## 典型應用場景
| 情境 | 怎麼用 |
|------|--------|
| 專案延遲 | 「是我低估了」而非「因為別人沒配合」 |
| 團隊失誤 | 身為領導承擔結果，同時內部公正檢討 |
| 個人承諾 | 無人監督時仍履行 |
| 社會議題 | 承擔作為公民的那份責任，不完全外包給政府 |

## 常見誤用
- 把「責任感」變成過度自責，包攬他人責任
- 只在成功時「負責」，失敗時推給環境
- 把責任感變成責備他人的道德武器
- 責任模糊：大家都「有責任」等於沒人負責
- 只對上負責，對下與對事不負責

## 如何自我練習
1. 每日反省：今天哪件事我本可多負一分責任？
2. 承諾前先估算真實能力，不為面子超承諾
3. 用「我語言」取代「他語言」：「我沒做好 X」取代「他們害我」
4. 建立「責任清單」，區分：我的、他人的、共同的、制度的

## 與其他 HC 的關係
- [[道德勇氣]]：承擔後果需要勇氣
- [[一致性]]：責任感需跨情境穩定
- [[自我覺察]]：準確區分責任歸屬需要自覺
- [[領導原則]]：領導核心之一是承擔集體責任
- [[制訂策略]]：責任分配是策略設計的一部分
