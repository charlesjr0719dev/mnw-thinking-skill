# 02 非語言溝通

## 子分類角色
非語言溝通處理「文字以外的所有傳遞通道」——視覺版面、聲音語調、肢體表情、媒介選擇、多模態整合。密涅瓦的 Multimodal Communications Cornerstone 本質上就是相信「溝通從來不只是文字」。

這個子分類的四個 HC 從「設計」到「整合」漸次擴張：[[溝通設計]] 建立視覺基本功、[[表達]] 訓練口語與肢體、[[媒介]] 提供選通道的策略、[[多媒體]] 則是將多個模態組合成整體體驗。

## HC 清單

| HC | 一句話摘要 |
|----|------------|
| [[溝通設計]] | 視覺版面、色彩、排版的構圖原則 |
| [[表達]] | 語調、停頓、眼神、手勢等口語 paralinguistic |
| [[媒介]] | 選擇最適合訊息性質的通道 |
| [[多媒體]] | 整合文字、圖像、聲音、影片的綜效設計 |

## 學習順序建議
1. 先練 [[媒介]]——選對通道是前提
2. 再練 [[溝通設計]]——視覺是多媒體的基礎
3. 搭配 [[表達]]——口語場合的非語言控制
4. 最後整合 [[多媒體]]——把前三者綜合運用
