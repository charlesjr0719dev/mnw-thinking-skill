# 01 語言溝通

## 子分類角色
語言溝通涵蓋「用文字與口語傳遞訊息」的所有技藝，從受眾分析、字詞選擇、句段構成到整體結構，再到專業形象的展現。這是所有書寫、簡報、對話、寫作的共同基礎——在決定「怎麼說」之前先決定「為誰、說什麼、如何組織」。

這個子分類的六個 HC 形成一條從「策略」到「工藝」的光譜：[[受眾]] 決定方向、[[論題]] 立定核心、[[組織]] 搭建骨架、[[構成]] 雕琢句子、[[意涵]] 打磨字詞、[[專業精神]] 統整整體氣質。

## HC 清單

| HC | 一句話摘要 |
|----|------------|
| [[受眾]] | 根據接收者調整訊息內容與形式 |
| [[意涵]] | 覺察字詞的言外之意與情感色彩 |
| [[構成]] | 句與段層級的清晰、流暢、經濟 |
| [[組織]] | 整體作品的結構與順序安排 |
| [[論題]] | 以一句明確可辯護的主張統攝全篇 |
| [[專業精神]] | 細節、一致性、尊重受眾的可信形象 |

## 學習順序建議
1. 先練 [[受眾]] 與 [[論題]]——它們決定後續一切決策
2. 再練 [[組織]] 與 [[構成]]——骨架與血肉的搭配
3. 最後精修 [[意涵]] 與 [[專業精神]]——細節打磨與氣質統整
