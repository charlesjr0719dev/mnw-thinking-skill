# 溝通設計（#composition，非語言版本／#design）

> 一句話定義：運用視覺版面、色彩、排版等設計原則，強化訊息傳遞效率。

## 密涅瓦怎麼定義
在非語言溝通子分類中，#composition（或稱 #design）指「視覺構圖」——如何安排畫面元素讓資訊被正確感知。屬於 Multimodal Communications 中非語言溝通的核心 HC，服務資訊的視覺可讀性。

## 核心概念
- **視覺階層**：用大小、粗細、色彩建立閱讀順序
- **對齊與留白**：格線對齊、充足留白比裝飾更能提升專業感
- **格式塔原則**：鄰近、相似、封閉、連續決定元素分組
- **色彩心理與對比**：色彩承載意涵，對比度保障可讀性（WCAG）
- **少即是多**：每張投影片／圖表一個核心訊息

## 典型應用場景
| 情境 | 怎麼用 |
|------|--------|
| 簡報投影片 | 每張一個重點，字數 ≤ 6-6 原則 |
| 資訊圖表 | 用空間位置編碼數據，而非純文字列表 |
| 履歷 | 分欄對齊、統一字級、關鍵字加粗 |
| 網頁／App | F 型或 Z 型視覺動線安排內容 |

## 常見誤用
- 投影片放滿文字，讀者分心讀字而不聽講
- 濫用色彩，每個元素都強調＝都不強調
- 裝飾性圖示反而造成雜訊（chartjunk）
- 忽略色盲與無障礙考量

## 如何自我練習
1. 用 Nancy Duarte 或 Edward Tufte 的範例分析頂級簡報
2. 每張投影片強迫自己砍到 50% 內容
3. 打印黑白版檢查對比度是否足夠
4. 模仿優秀海報／資訊圖表的版面結構

## 與其他 HC 的關係
- [[多媒體]]：設計是多媒體組合的基礎
- [[組織]]：視覺結構與文本組織互為表裡
- [[受眾]]：不同文化對色彩、閱讀方向有不同預期
- [[表達]]：現場簡報時設計服務於口語表達
