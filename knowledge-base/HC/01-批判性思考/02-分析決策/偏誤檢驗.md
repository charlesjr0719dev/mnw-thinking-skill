# 偏誤檢驗（#biasidentification）

> 一句話定義：辨識當前判斷中具體是哪些認知偏誤正在扭曲你的思考。

## 密涅瓦怎麼定義

Minerva 將 #biasidentification 定義為「辨識特定認知偏誤在當下決策或推論中的運作方式」。屬於 **Formal Analyses（#formalanalyses）** 課程中與 Kahneman 系統一/系統二思維相關的 HC。重點不是背偏誤清單，而是練到能在當下或事後指出「這裡是錨定」、「這裡是可得性」。

## 核心概念

- **常見偏誤家族**：確認偏誤、可得性啟發、錨定效應、代表性啟發、過度自信、存活者偏誤、事後諸葛、基本歸因謬誤。
- **偏誤是系統一的捷徑**：不是錯誤，是快速思考的副產品，用對場合反而高效。
- **辨識分三層**：知道有這偏誤 → 知道它何時觸發 → 能在自己身上即時辨識。
- **他人偏誤好抓，自己偏誤難抓**（bias blind spot 本身就是偏誤）。

## 典型應用場景

| 情境 | 怎麼用 |
|------|--------|
| 面試決策 | 辨識「首因效應」讓前五分鐘印象主導評價。 |
| 投資判斷 | 看到朋友賺錢的股票想跟進——可得性啟發。 |
| 專案規劃 | 估工期永遠低估——規劃謬誤（planning fallacy）。 |
| 新聞閱讀 | 注意媒體選題讓你誤以為某類事件很常見。 |

## 常見誤用

- **偏誤標籤濫用**：把任何不同意的觀點都貼上「確認偏誤」。
- **背清單不練識別**：能背 100 個偏誤，但自己做決策時一個都沒察覺。
- **以為知道就免疫**：研究顯示知道偏誤並不等於不犯，只是容易事後辨識。
- **只檢別人不檢自己**：辨識他人偏誤成為優越感來源。

## 如何自我練習

1. 每週寫一個決策日記，事後標出可能涉及的偏誤。
2. 做決策前強迫自問三個偏誤：「我是不是錨在某個數字？」「我是不是只看符合我的證據？」「我是不是高估自己掌握？」
3. 跟人討論時把「我覺得我現在可能在 X 偏誤」說出聲，降低盲點。
