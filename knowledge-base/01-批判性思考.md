# 01 批判性思考（Thinking Critically）

> 一句話定義：能夠系統性地分析問題、檢驗推理、評估主張與決策的心智能力。

## 在 Minerva 四大能力中的定位

Minerva 大學將全人能力拆成四大核心：**批判性思考**、創造性思考、有效溝通、有效互動。批判性思考排在第一位，因為它是其他三者的前提——無法辨識錯誤推理與脆弱證據的人，既無法做出好決策，也無法有效溝通自己的判斷。

這一能力主要在 Cornerstone Course 中的 **Formal Analyses（形式分析）** 與 **Empirical Analyses（經驗分析）** 兩門課教授。Minerva 官方文件將批判性思考拆成四個子領域：分析問題、權衡決策（analyzing inferences / weighing decisions）、分析資料、評估推理、評估主張。本資料夾將其整理為五個更貼近實用的子分類。

## 五個子分類

- [[01-分析問題]]：在開始解之前，先確認這是不是**對的問題**。包含 #rightproblem、#breakitdown、#gapanalysis、#variables、#gametheory。
- [[02-分析決策]]：理解自己與他人如何做選擇，並引入決策輔助工具。包含 #psychophysics、#purpose、#utility、#biasidentification、#biasmitigation、#decisiontrees。
- [[03-分析資料]]：用統計與機率的語言描述世界。包含 #descriptivestats、#probability、#distributions、#confidenceintervals、#correlation、#regression、#significance。
- [[04-評估理由]]：檢視支持一個主張的推理與證據品質。包含 #evidencebased、#sourcequality、#deduction、#induction、#fallacies。
- [[05-評估主張]]：在接受任何觀點前，先通過合理性、可驗證性與脈絡的檢驗。包含 #interpretivelenses、#context、#critique、#plausibility、#testability、#estimation。

## 整體學習路徑建議

1. **從「問對問題」開始**：所有後續工具都建立在正確問題之上。先讀 [[01-分析問題]]。
2. **再學評估，後學分析**：先會辨識爛推理（[[04-評估理由]]、[[05-評估主張]]），再學如何自己產生好推理（[[03-分析資料]]、[[02-分析決策]]）。這樣才不會把自己也訓練成同樣的錯誤。
3. **數據素養是地基**：沒有機率、分布、相關性的基礎概念，絕大多數公共討論都只能停留在直覺。[[03-分析資料]] 應優先拿下。
4. **決策與偏誤放最後**：偏誤清單人人會背，但要在壓力下真的用出來需要前面所有工具當後援。
5. **每個 HC 至少找三個真實情境練習**：Minerva 的設計原意是「可以在畢業後的日常生活中用出來」，不是考試用的。
